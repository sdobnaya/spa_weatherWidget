1. Для установки зависимостей выполни:

```sh
yarn
# или
npm i
```

2. Для запуска проекта выполни:

```sh
yarn dev
# или
npm run dev
```

3. Для сборки проекта выполни:

```sh
yarn build
# или
npm run build
```

4. Для запуска проекта в режиме `production` выполни:

```sh
yarn build && yarn start  
# или
npm run build && npm run start
```
